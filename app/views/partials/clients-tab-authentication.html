JSON Web Token (JWT)

JWT authentication is enabled. JWTs are used to securely transmit information between a client and a server as a JSON object. The information can be trusted because it is digitally signed using a secret or public/private key pair. The details of this auth mechanism is configured on the OpenHIM Core.
